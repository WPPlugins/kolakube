== Description ==

A functionality plugin that enhances Kolakube products like Marketers Delight and Page Leads. Adds a simple API you can use to build admin pages and meta boxes. More detailed documentation coming soon. To see this plugins power, check out the Page Leads suite from Kolakube.

== Installation ==

1. Upload `kolakube` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Done! This plugin doesn't do anything by itself
